# タスク管理Webアプリ

## Description
**現在開発中**<br>
個人向けタスク管理Webアプリ<br>
**PWA・TWA化済み**<br>
タスクをこなすと**地球**が育つ<br>

| ------------- | ------------- |
| vue  | 2.6.11 |
| vuex  | 3.4.0 |
| vue-router | 3.2.0 |
| core-js | 3.6.5 |
| firebase | 7.16.1 |
| vuetify | 2.4.5 |
| vue-loading-template | 1.3.2 |

## Getting Started
1. `npm run serve`<br>

or
1. `vue ui`
2. task > serve > run task
